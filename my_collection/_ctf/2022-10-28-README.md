# README

其实在去年的这个时候，就应该有这类的博客了，但可惜刚学完汇编不久，逆向也只是课上接触了一点点，就算去做校巴，也很难一下就上手。不管是`crypto`还是`rev`，总觉得很难从课堂迁移到实操中。后来在暑假，有人提供了一些工具和基本的方法，逐渐摸索的入门了以后，才觉得真正可以做得下去，慢慢发现了`rev`和`pwn`的一些乐趣和技巧，也后来能在校巴上做题，以及参与上周的校赛

也因为觉得这件事情有趣而且有其意义了，才决定专门开一个博客网站，每天学习，每天写作，用来督促自己罢
